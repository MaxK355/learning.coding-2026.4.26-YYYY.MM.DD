## 题目描述

你有若干个区间，表示为 $[a_i, b_i]$，其中 $a_i, b_i$ 为整数。

要求选若干个点，使得每个区间中至少有一个点（不同的区间允许共用一个点），求最少需要的点数。

## 输入格式

第一行是一个整数 $n$，表示区间个数。

接下来 $n$ 行，每行 2 个整数 $a_i, b_i$，表示一个区间 $[a_i, b_i]$。

## 输出格式

一个整数，表示所求的最少点数。

## 输入样例 #1

```
6
1 2
2 3
5 6
3 4
4 5
6 7
```

## 输出样例 #1

```
3
```

## 样例 #2

## 输入样例 #2

```
5
1 4
2 5
3 6
6 8
7 9
```

## 输出样例 #2

```
2
```

## 提示

$1 \le n \le 200000$。
