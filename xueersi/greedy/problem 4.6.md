# 湖边的推销员

## 题目描述

一个湖边有 N 户人家，沿着湖边环湖一周的路程是 K 米。第 i 户的位置是，从湖最北端顺时针走 Aᵢ 米的位置。每户之间的移动只能沿着湖边进行。

推销员可以从任何一家出发，他的目的是访问湖边的所有 N 户人家，求出推销员需要走最少路程。

## 输入格式

第 1 行，两个正整数 K, N，代表环湖一周长度和家的个数，以空格分隔。

第 2 行，N 个正整数 A₁, A₂, ⋯, Aₙ，以空格分隔。

## 输出格式

输出 1 个整数，推销员需要走最少路程。

## 样例

**输入样例#1**
```
20 3
1 2 10
```

**输出样例#1**
```
9
```

**输入样例#2**
```
20 3
0 5 15
```

**输出样例#2**
```
10
```

## 样例说明

样例1：
- 从第1家出发的最短路线为 1→2→3，总路程 9。
- 从第2家出发的最短路线为 2→1→3，总路程 10。
- 从第3家出发的最短路线为 3→2→1，总路程 9。
从第1家或第3家出发总路程最短。

样例2：
- 从第2家出发的最短路线为 2→1→3，总路程 10，是最短的。

## 数据说明

2 ≤ K ≤ 10⁶

2 ≤ N ≤ 2×10⁵

0 ≤ A₁ < A₂ < ⋯ < Aₙ < K
