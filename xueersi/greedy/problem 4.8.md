# [USACO05MAR] Yogurt factory 机器工厂

## 题目描述

小 T 开办了一家机器工厂，在 N 个星期内，原材料成本和劳动力价格不断起伏，第 i 周生产一台机器需要花费 Cᵢ 元。若没把机器卖出去，每保养一台机器，每周需要花费 S 元，这个费用不会发生变化。

机器工厂接到订单，在第 i 周需要交付 Yᵢ 台机器给委托人，第 i 周刚生产的机器，或者之前的存货，都可以进行交付。

请你计算出这 n 周时间内完成订单的最小代价。

## 输入描述

第一行输入两个整数 N 和 S，接下来 N 行每行两个数 Cᵢ 和 Yᵢ。

## 输出描述

输出一个整数，表示最少的代价。

## 样例

**样例1**

输入
```
4 5
88 200
89 400
97 300
91 500
```

输出
```
126900
```

## 提示

1 ≤ n ≤ 10⁴，1 ≤ Cᵢ ≤ 5000，1 ≤ S ≤ 100，0 ≤ Yᵢ ≤ 10⁴。
