# 题目描述

黑心企业的老板黑先生只愿意雇临时工来完成工作。

每项工作都有一个难度 $w$，每个临时工都有一个能力值 $p$，当且仅当 $w \le p$ 时，这个临时工可以完成这项工作。一个临时工最多只能完成一项工作，而且无论工作的难度如何都会收取固定的工资 $k \times p$，其中 $p$ 是他的能力值，$k$ 是一个事先给定的数。

黑先生这里有 $m$ 项工作需要完成，他拿到了一份 $n$ 名可用的临时工能力清单，请帮他算一算，他最少花多少钱才能把所有工作完成吧。

# 输入描述

输入共 3 行：

- 第 1 行：3 个正整数 $n, m, k$，分别为可用临时工数、工作个数和计算临时工工资用的系数；
- 第 2 行：用空格隔开的 $n$ 个正整数 $p_1, p_2, \ldots, p_n$，分别为每名临时工的能力值，保证这些临时工一定能完成所有工作；
- 第 3 行：用空格隔开的 $m$ 个正整数 $w_1, w_2, \ldots, w_m$，分别为每项工作的难度；

# 输出描述

输出共 1 行：

- 第 1 行：1 个正整数，为黑先生最少需要付的工资。

# 样例 1

**输入**

```
4 3 2
100 8 7 6
1 2 99
```

**输出**

```
226
```

# 提示

- $1 \le m \le n \le 300000$
- $1 \le k \le 10$
- $1 \le p_1, p_2, \ldots, p_n, w_1, w_2, \ldots, w_m \le 10^9$
